## What Are Web Application Components?
Web application components are the building blocks of a web application.
Each component has a specific role.

## Frontend Component
The frontend is what users see and interact with.
It includes pages, forms, buttons, and layouts.

## Backend Component
The backend processes requests from users.
It handles authentication, business logic, and data validation.

## Database Component
The database stores application data such as user details and content.
It plays a critical role in application functionality.

## Why This Matters for Security
Understanding components helps identify where attacks may occur.
It also helps developers and testers apply security controls correctly.
