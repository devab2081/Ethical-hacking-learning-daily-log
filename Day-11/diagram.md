# Day 11 – Vulnerability Assessment (Visual Diagram)

## What is Vulnerability Assessment?

```
System / Application
|
v
Identify Weaknesses
|
v
Analyze Risk Level
|
v
Recommend Fixes

```
---

## Vulnerability vs Threat

```
Vulnerability
│
├── Security weakness
├── Exists in system
└── Can be fixed

Threat
│
├── Potential attacker
├── May exploit weakness
└── Causes damage

```
---

## Purpose of Vulnerability Assessment

```
Find Weaknesses
|
v
Prioritize Risks
|
v
Fix Issues Early
|
v
Improve Security

```
